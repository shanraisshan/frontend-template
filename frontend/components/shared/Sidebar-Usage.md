# Sidebar Usage

**Last Updated**: 2025-11-23

This component is used in:
- `app/RootLayoutClient.tsx` - Global sidebar navigation wrapper

## Component Features
- Slides in from the left with smooth animation
- Backdrop overlay that closes sidebar when clicked
- Navigation items: Home (/), Feature (/feature), Settings (/settings)
- Active route highlighting with blue left border
- Fixed position with z-index layering
- Width: 250px
- Smooth transitions (0.3s ease-in-out)
- Version information displayed at bottom (Version 1.0 with deployment date)
- Flexbox layout to push version info to bottom
- Version date placeholder for CI/CD updates
