# GlobalHeader Usage

**Last Updated**: 2025-11-23

This component is used in:
- `app/RootLayoutClient.tsx` - Global header wrapper for all pages

## Component Features
- Fixed position at top of screen (50px height)
- Contains HamburgerMenu for sidebar navigation
- Appears on ALL pages (home, settings, and future pages)
- White background with bottom border
- z-index: 100 to stay above page content
- Global navigation element

## Purpose
Provides consistent navigation access across the entire application. The hamburger menu allows users to access the sidebar from any page, making it a true global navigation solution.
